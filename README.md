# Exemple d'application Web avec Thymeleaf

## Prérequis
- JDK 1.8
- MAVEN 3.1

## Execution du site localement
```
	git clone https://github.com/soprasteria/thymeleaf-example.git
	cd thymeleaf-example
	./mvn jetty:run
```

Vous pouvez ensuite accéder au site ici : http://localhost:8080/
